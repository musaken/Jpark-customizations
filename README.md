# Jemspark Customizations

Custom frontend and WooCommerce enhancements for jemspark.co.ke.

## Includes
- Homepage Ah battery filter
- Shop Ah battery filter
- Sticky battery capacity bar
- Glassmorphism UI styles
- WooCommerce layout tweaks
- Custom snippets for Elementor / Code Snippets plugin

## Site Stack
- WordPress
- WooCommerce
- Elementor
- Partdo child theme

## Folder Structure

- `css/` = custom styles
- `js/` = JavaScript functionality
- `html/` = ready-to-paste HTML blocks
- `php/` = WooCommerce hooks and snippets
- `notes/` = where each code block should be pasted
